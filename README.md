## 安装

### 从 PyPI 安装（推荐）

- 使用 nb-cli  

```sh
nb plugin install nonebot-plugin-tty
```

- 使用 pdm

```sh
pdm add nonebot-plugin-tty
```

- 使用 pip

```sh
pip install nonebot-plugin-tty
```

### 从 GitHub 安装（不推荐）

```sh
pdm add git+https://github.com/nonepkg/nonebot-plugin-tty
```
